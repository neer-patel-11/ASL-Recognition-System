🤟 ASL Recognition System

This application helps you identify American Sign Language (ASL) letters using image input.

📌 How to Use

  • Go to the Prediction Page
  • Click on "Upload Image"
  • Select or drag & drop an image of a hand sign
  • Click "Identify Sign"
  • View the predicted ASL letter and confidence score

📊 Understanding Results

  • Predicted Letter: The ASL alphabet detected
  • Confidence: How sure the model is (%)
  • Top Predictions: Other possible matches
  • Latency: Time taken for prediction

⚙ Admin Panel

  • Go to Pipeline Admin
  • View logs, model metrics, and experiments
  • Run test suite for system validation

⚠ Tips

  • Use clear hand images with good lighting
  • Avoid blurry or dark images
  • Ensure only one hand sign is visible
Better image quality = better prediction accuracy.

❓ Troubleshooting

  • If prediction fails, try another image
  • Check internet/server connection
  • Ensure file format is PNG/JPG/WEBP
